# Spicy-Lyrics-custom-background

This project is based on the original Spicy Lyrics extension by Spikerko.

The original project, architecture, and foundation belong to the original developers. This repository is not intended to replace their work but to build upon it with a custom visualizer experience.

## What is different?

This version focuses on creating a more immersive fullscreen music experience inspired by:

* Tame Impala live visuals
* Psychedelic concert lighting
* Wallpaper Engine fluid visualizers
* Vocal-reactive star fields
* Dynamic album-cover based lighting

The goal was never to create a traditional audio visualizer.

Instead, the objective is to make the album artwork itself feel alive.

## Main Features

### Dynamic Wave Illumination

The original fluid background remains the visual foundation.

Music energy influences the existing cover waves and color regions, allowing them to glow and breathe with the song while preserving the original Spicy Lyrics aesthetic.

### Vocal-Reactive Stars

An optional star layer can react to vocals and high notes.

The stars are designed to feel atmospheric and musical rather than distracting:

* subtle twinkling
* vocal-driven shimmer
* local color adaptation
* multiple star styles

### Custom Presets

Includes custom visual styles inspired by different listening experiences:

* Vanilla
* Smooth
* Cinematic
* Energetic
* Insane
* Tame Impala / Eventually
* Strokes / Snow Strippers

### Extended Controls

Additional settings allow users to customize:

* movement speed
* wave glow strength
* impact intensity
* brightness reactivity
* star appearance
* performance behavior
